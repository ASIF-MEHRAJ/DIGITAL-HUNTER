## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox, Grid, Custom Animations)
* **Typography:** Orbitron & Montserrat via Google Fonts
* **Icons:** Remix Icons / FontAwesome
* **Styling Techniques:** Glassmorphism, Neon Glow Effects, Multi-stop Gradients

## ✨ Key Features
* **Hero Section:** High-impact typography with custom neon call-to-action buttons.
* **Glossy Stats Bar:** A "metallic" glassmorphic bar displaying studio achievements with backdrop-filter effects.
* **Interactive Project Cards:** Hover-animated cards featuring unique light-leak borders and "bloom" glows.
* **Process Roadmap:** A clean, vertical timeline showcasing the 5-stage development cycle from Discussion to Launch.
* **Futuristic Navigation:** Minimalist glass-textured header with smooth hover states.
